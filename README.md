# Data Structures and Algorithms Lab

A comparative study of algorithms, data structures and memory models across multiple languages.

## Goals

- Implement common data structures and algorithms manually.
- Compare different memory models across languages.
- Study time and space complexity.
- Understand trade-offs between manual memory, garbage collection and ownership.
- Practice problem solving using multiple languages.
- Benchmark selected implementations.

## Languages

- C
- C++
- TypeScript
- Java
- Python
- Go
- Rust

## Topics

- Big O notation
- Time complexity
- Space complexity
- Recursion
- Stack vs heap
- Garbage collection
- Ownership
- Arrays
- Linked lists
- Stacks and queues
- Hash tables
- Trees
- Graphs
- Sorting
- Searching
- Dynamic programming
